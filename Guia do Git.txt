Na pasta onde deseja criar um projetos ou adicionar códigos:
-> git init

Crie um arquivo qualquer.

Para que o git monitore as alterações ou salve.

-> giit add "nome do arquivo"

ou se for mais de um arquivo:

-> git add -A

Para verificar a status dos arquivos:

-> git status

Para salvar as alterações no repositório local:

-> git commit -m "mensagem"

Para ver um relatório das alterações no repositósio e quem fez os Commits:

-> git log

Como faz pra saber em que Branch a gente está modificando o código?

-> git branch

Este comando lista os Branchs e o asterisco (*) indica em qual Branch estamos atualmente.

Adicionar um repositório remoto

-> git remote add origin url

:origin -> é o nome do repositório

:url -> é o https (link do repositório)

-> git remote -v

para receber os arquivos do repositório remoto:

-> git fetch -u origin master

para enviar os arquivos do repositório local (do computador do usuário) para o repositório remoto:

-> git push -u origin master

:origin -> nome do repositório que irá receber os arquivos

:master -> de qual branch será enviado os arquivos (repositório local)
